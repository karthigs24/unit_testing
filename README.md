Sparkout CRM Node Backend Boilerplate
==================================

Getting Started
---------------

```sh
# clone it
git clone https://github.com/gotocva/sparkout-crm.git
cd sparkout-crm

Copy .env.example into .env

# Install dependencies
npm install

# Start development live-reload server
npm run dev

# Start production server:
npm start

```
